# Ampliació Base de Dades P2

Exercises completed in class and exam-related solutions for the second part of the course, focused on the design and implementation of patterns.

## About the Repository

This repository contains exercises, practice problems, and exam solutions developed during the course.  
The goal is to provide a reference for understanding and applying software design patterns used in the subject.

The solutions included here are based on:

- Class corrections and annotations from the professor
- Solutions and ideas from previous students and public repositories:
  - ABBDDEP Repository: https://github.com/JosepAntoni/ABBDDEP/tree/master
  - Design Patterns in Java: https://github.com/mrg20/design-patterns-in-java/tree/master/src
  - ABBDD Examens Resolts: https://github.com/didicayu/ABBDD-Examens-Resolts/tree/main
- AI-assisted explanations and comparisons to help understand the problems and reach a more consistent interpretation of the solutions

> The objective is educational: understanding the reasoning behind the patterns and implementations rather than providing “perfect” solutions.

## Technologies

- Java
- IntelliJ IDEA

The project was mainly developed and tested using IntelliJ IDEA.

## Collaboration

Contributions, corrections, and alternative implementations are welcome.

You can collaborate by:

- Opening an issue for mistakes or improvements
- Creating a fork of the repository
- Submitting a pull request with fixes or new solutions

Different approaches to the same exercise are appreciated, especially if they improve readability, architecture, or pattern usage.

## Disclaimer

Some exercises may have multiple valid implementations depending on interpretation or design decisions.  
This repository represents a consensus built from class material, previous student solutions, and additional research it may have a better implementations or it can change in the future, so you are free to collaborate.
